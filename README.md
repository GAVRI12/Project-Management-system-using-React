# Project Management Dashboard

A simple Project Management Dashboard developed using React JS.  
The application allows users to add tasks and update their status as Pending, In Progress, or Completed through a clean and simple user interface.

---

##  Aim
To develop a simple dashboard for project management using React JS.

---

##  Problem Statement
Develop a web-based dashboard where users can manage project tasks, view their current status, and update task progress efficiently.

---

##  Features
- Add new tasks
- View all tasks in a dashboard
- Update task status (Pending / In Progress / Completed)
- Simple and user-friendly interface

---

##  Technologies Used
- React JS
- JavaScript
- HTML
- CSS
- GitHub (Version Control)
